设计模式  
====

## 创建型模式
* 工厂方法模式
	* 普通工厂
	* 多个工厂方法
	* 静态工厂
* 抽象工厂模式
* 单例模式
	* 非线程
	* 线程 同步锁
	* 线程 内部类 解决同步锁异常问题
	* 线程 同步锁 解决同步锁异常问题
* 建造者模式
* 原型模式

## 结构型模式
* 适配器模式
	* 类的适配器模式
	* 对象的适配器模式
	* 接口的适配器模式
* 装饰器模式
* 代理模式
* 外观模式
* 桥接模式
* 组合模式
* 享元模式

## 行为模式
* 父类与子类
	* 策略模式
		* 策略模式能有效的减少if-else语句
	* 模板方法模式
* 两个类之间
	* 观察者模式
	* 迭代子模式
	* 责任链模式
	* 命令模式
* 类的状态
	* 备忘录模式
	* 状态模式
* 通过中间类
	* 访问者模式
	* 中介者模式（只有一个仲裁者）
	* 解释器模式（Interpreter）
